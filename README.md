# THRED

**THRED** is a visual-first note-taking and planning app that lets you connect ideas as nodes in an interactive 3D graph, organize them with tags and branches, and turn them into actionable tasks with reminders.

---

##  Features

-  **Graph-Based Thinking**
  - Create notes as nodes in a 3D space
  - Connect ideas visually

-  **Smart Connections**
  - Search and link nodes easily
  - Create and connect in one flow

-  **Branching System**
  - Explore alternate paths from a single idea
  - Keep different directions organized

-  **Tags & Filters**
  - Tag nodes for better organization
  - Filter and focus on what matters

-  **Global Search**
  - Instantly find nodes by title or content
  - Jump directly to them in the graph

-  **Tasks & Planning**
  - Turn nodes into tasks
  - Add due dates and track progress

-  **Notifications**
  - Get reminders for upcoming and overdue tasks

-  **Timeline View**
  - View tasks by date (Today, Upcoming, Later)

-  **Auto Layout**
  - Organize your graph into a readable structure

-  **Offline First**
  - Data stored locally using IndexedDB

-  **Export / Import**
  - Backup your data as JSON
  - Restore anytime

-  **Graph Snapshot Export**
  - Export your current view as an image

##  Getting Started

git clone https://github.com/your-username/THRED.git
cd THRED

Build for Android (Capacitor)
npm install
npx cap add android
npx cap open android

Build and run from Android Studio.

 How It Works
Each idea = a node
Nodes can connect to other nodes
Nodes can branch into alternate paths
Nodes can become tasks with deadlines

Think → Connect → Plan → Execute
